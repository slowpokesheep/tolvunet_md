# Markdown files for presentation

## Install
* [Github/reveal-md](https://github.com/webpro/reveal-md)
  * `npm install -g reveal-md`

## Usage

```
git clone https://github.com/slowpokesheep/tolvunet_md.git
cd markdown
npm install
npm start
```
